这是一款由python编写的回合制战斗小游戏

我们有如下优点

  • 全在脑子里想象所以非常高的帧率
  • 完全开源可自定义
  • 有着不愿透露姓名的美工写的优秀文案(为其留了彩蛋)
  • 多平台、多语言支持(翻译中！)

如何玩

  • Linux
    直接运行start.sh
  • windows
    直接运行start.bat
  • Android
    下载termux
    通过pkg install python命令安装python
    给termux储存权限
    cd到安装目录
    输入sh start.sh运行
  • 其他系统(如IOS,MACOS)请直接在对应IDLE中运行main.py
    当然ios可能没有IDLE
